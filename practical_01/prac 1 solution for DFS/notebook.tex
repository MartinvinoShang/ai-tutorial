    Here the path was {[}'A', 'D', 'F', 'G', 'I', 'K', 'C', 'B', 'E',
'J'{]}.

Looking at the picture of the graph above we can see it might have a
problem because this path doesn't follow a logical path (e.g. no edge to
go from K to C).

The reason for this is because the path that is output also shows
backtracking steps. The actual path that DFS followed is: - go from
A-D-F-G-I, - this is a dead end so go back to F and follow F-K - another
dead end so back to the next branch and follow A-C - finally follow the
path A-B-E-J
